# ⚖️ Legal Document Analyzer

An AI-powered tool that extracts key information from legal documents such as parties, dates, and financial details.

## 🚀 Features
- Extracts names, dates, and amounts
- Identifies key sections in documents
- Fast and simple processing

## 🛠️ Tech Stack
- Python
- NLP
- Regex

## ⚙️ How to Run

```bash
git clone https://github.com/shawnmendes123/legal-doc-analyzer
cd legal-doc-analyzer
pip install -r requirements.txt
python main.py
